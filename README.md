# Swingsonic

A translation API to allow [Subsonic](https://subsonic.org/pages/index.jsp) clients to work with [Swing Music](https://github.com/swing-opensource/swingmusic).

## Endpoints

- getPlaylists*
- getAlbumList2
- getAlbumList
- getCoverArt
- stream
- download
- getArtists
- getArtist
- getLyrics
- search3
- ping
